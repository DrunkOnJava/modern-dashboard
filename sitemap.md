## Main Routes

### Dashboard (/)

- Main dashboard interface
- Customizable dashboard grid
- Widget management
  - Add/remove cards
  - Card settings
  - Layout customization
- Dark mode toggle
- Keyboard shortcuts
- Analytics integration

- User profile management
- Preference settings
- Account customization

### Authentication (/auth)

- Login (/auth/login)
  - Email/password login
  - Google sign-in
  - Password recovery
- Registration (/auth/register)
  - New account creation
  - Email verification
